# v0.7.3 — mermaid pan/zoom canvas

Long mermaid diagrams used to expand to their natural rendered height —
the user-reported bug-report flowchart hit ~1100 px tall in the preview
pane, dominating vertical scroll and pushing surrounding markdown
off-screen.

v0.7.3 bounds every rendered mermaid block inside a fixed-height viewport
(60vh capped at 480px) and turns the diagram itself into a pan/zoom canvas,
matching the UX of [mermaid.live](https://mermaid.live).

## What's new

- **Bounded viewport** — every `.mermaid-block` is now ~480px tall max
  with `overflow: hidden`. Diagrams larger than the viewport are scrollable
  via pan, not via the parent preview pane.
- **Drag to pan** — click anywhere inside the diagram and drag.
- **Ctrl/Cmd+wheel to zoom** — plain wheel still scrolls the preview pane
  past the diagram (Google-Maps-style modifier-gated zoom).
- **Toolbar overlay** — top-right of every diagram:
  - **−** zoom out
  - **+** zoom in
  - **⟲** reset / fit to view
  - **↗** open in mermaid.live (full editor + unlimited canvas)
- **Theme-aware** — toolbar + wrapper adapt to the preview pane's
  light/dark toggle (introduced in v0.7.0).

## Under the hood

- Added `svg-pan-zoom@^3.6.2` (~8 KB gzip). Lazy-loaded alongside mermaid
  itself so docs without mermaid blocks pay zero cost.
- `useMaxWidth: false` retained from v0.7.1 so foreignObject heights stay
  accurate (multi-line labels still render cleanly post-v0.7.2 fix).
- Morphdom cache contract preserved: unchanged mermaid blocks across
  keystrokes keep their attached svg-pan-zoom instance — pan/zoom state
  survives debounced re-renders. Changed blocks trigger
  `_panzoom.destroy()` via the new `onBeforeNodeDiscarded` callback so
  global event listeners don't leak.

## Known limitations

- Theme switch re-renders mermaid → pan/zoom state resets (matches
  mermaid.live behaviour).
- Touch gestures (pinch-zoom on tablets) deferred — desktop-first.
  Re-add `hammerjs` integration later if requested.

## Files

- `src/lib/mermaid.ts` — pan/zoom wiring + toolbar builder + mermaid.live
  URL encoder.
- `src/components/Preview.svelte` — morphdom `onBeforeNodeDiscarded` hook.
- `src/app.css` — bounded `.mermaid-block` + `.mermaid-toolbar` styles
  (light + dark).
- `src/svg-pan-zoom.d.ts` — TypeScript shim (svg-pan-zoom v3 ships no
  types).
- `package.json` — `+svg-pan-zoom@^3.6.2`, bump 0.7.2 → 0.7.3.
