Asset proxy userscript

A 50-line Tampermonkey companion that lets gh-md-editor render private/org github.com/user-attachments images inline.

Status: detecting…

1. Install Tampermonkey

If you don't already have a userscript manager: Get Tampermonkey →

2. Install this userscript

Tampermonkey auto-intercepts URLs ending in .user.js and shows its install dialog:

Install asset proxy userscript →

3. Reload the editor

Reload the gh-md-editor tab. github.com/user-attachments/... images now resolve cookie-authenticated through your browser. No backend, no PAT.

How it works

The page sends a postMessage to the userscript with the URL. The userscript runs GM.xmlHttpRequest with cookies attached — GitHub redirects to a short-lived S3 presigned URL — downloads the bytes, creates a blob: URL, and posts it back. The page sets <img src=blob:...>. Bytes never leave your browser.
