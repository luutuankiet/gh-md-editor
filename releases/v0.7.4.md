# v0.7.4 — plain-wheel zoom for mermaid canvas

v0.7.3 gated wheel-zoom on Ctrl/Cmd to keep the preview pane's wheel-scroll working uninterrupted. User feedback: prefer mermaid.live's plain-wheel behavior. One-line change.

## What's new

- **Wheel zoom is now plain-wheel** — no modifier needed. Wheel up = zoom in, wheel down = zoom out (matches the toolbar's − / + order and Google Maps / Figma / draw.io conventions).
- Zooms at cursor position (was already the v0.7.3 behavior for Ctrl/Cmd+wheel; carried over).
- Toolbar button tooltips updated: "Zoom in (or wheel up)" / "Zoom out (or wheel down)".

## Trade-off

With the cursor over a diagram, the wheel is captured — you can't scroll the preview pane past the diagram by wheeling over it. To scroll past:

- Move the cursor outside the diagram
- Use the preview pane's scrollbar
- Use keyboard arrow keys / Page Up/Down

This matches mermaid.live's behavior. v0.7.3's Ctrl/Cmd-gated approach was the alternative UX; if it's missed, easy 1-line flip via the modifier check.
